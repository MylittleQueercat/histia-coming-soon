# Histia — Cas pratique : Page "En Construction"

Implémentation de la page "En Construction" en Next.js, avec internationalisation FR/EN via l'URL.

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS

## Lancer le projet

```bash
npm install
npm run dev
```

## URLs

| URL | Langue |
|-----|--------|
| `/fr/coming-soon` | Français |
| `/en/coming-soon` | Anglais |

## Fonctionnalités

- Internationalisation FR/EN via le segment d'URL `[locale]`
- Mascotte et textes adaptés selon la langue
- Responsive (1400×900 et 1920×1080)
- Sidebar non implémentée (optionnelle selon les consignes)
